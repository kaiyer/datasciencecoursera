Prediction of Exercise Quality Using Machine Learning

Executive Summary

The goal of this modeling project is to develop a machine learning algorithm and train a model using a training dataset. The dataset consists of data from accelerometers on the belt, forearm, arm, and dumbell of 6 participants. They were asked to perform barbell lifts correctly and incorrectly in 5 different ways. The model predicts how well they did on the trials based on a categorical measure called classe ranked _{A, B, C, D}

Machine Based Learning

Two data sets were provided A training data set with 19,622 observations and a test set of 20 observations for which the student has to predict the categorical measure called classe.

A GBM algorithm in the caret package in R was used to train and validate the model. The training data was broken into a training and test data set where the outcomes of classe were available for the test data. The training data had 13737 observations and the test data had 5885 observations.

Data Modeling

The data was cleaned in excel and 29 variables including the outcome were chosen. The cleaned data was then loaded in R and boxplots of each covariate with the outcome was examined to visually inspect which covariates showed any response with the output classification. Based on this 28 variables were selected. The co-variates with correlation over 0.9 were dropped from the dataset. The data set for the input model had 25 covariates and one outcome. The data was also centered and scaled.

Model Fitting

The training dataset was then used to train the gbm model. The summary output of the model showed the model fitting worked well. Notice only Kappa and Accuracy are reported for multi-Classification tree-boosting algorithm outputs in gbm and caret.
